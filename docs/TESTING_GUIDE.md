## 🐛 常见问题排查

### 问题 1: npm install 失败
```bash
# 清除缓存
npm cache clean --force

# 删除 node_modules 重新安装
rm -rf node_modules package-lock.json
npm install
```

### 问题 2: 端口冲突
```bash
# 检查端口占用
netstat -ano | findstr :5173
netstat -ano | findstr :8000

# 杀掉占用进程（Windows）
taskkill /PID <PID> /F
```

### 问题 3: 后端 CORS 错误
确保后端 `main.py` 包含：
```python
app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:5173"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

### 问题 4: TypeScript 错误
```bash
# 跳过类型检查直接运行
npm run dev -- --force
```

### 问题 5: 页面空白
1. 打开浏览器控制台（F12）
2. 查看 Console 错误信息
3. 查看 Network 请求状态
