### 常见问题

#### 问题 1: 页面空白
**原因**: 可能是 JavaScript 错误

**解决**:
1. 按 F12 打开浏览器控制台
2. 查看 Console 标签的错误信息
3. 截图发给我分析

#### 问题 2: API 请求失败
**原因**: 后端未启动或端口不对

**解决**:
1. 检查后端是否在 8000 端口运行
2. 访问 http://localhost:8000/health 测试
3. 查看浏览器 Network 标签的请求状态

#### 问题 3: 样式错乱
**原因**: CSS 未加载

**解决**:
1. 硬刷新页面（Ctrl+Shift+R）
2. 清除浏览器缓存
3. 重启开发服务器

#### 问题 4: 无法发送消息
**原因**: 后端 CORS 配置

**检查后端 `main.py`**:
```python
app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:5173"],  # 确保包含前端地址
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```
